subroutine adios_helpers_definitions_mod::define_adios_scalar::define_adios_double_scalar ( integer(kind=8), intent(in)  adios_group,
integer(kind=8), intent(inout)  group_size_inc,
character(len=*), intent(in)  path,
character(len=*), intent(in)  name,
real(kind=8), intent(in)  var 
)
private

Define an ADIOS scalar double precision variable and autoincrement the adios group size by (8).

Parameters
adios_groupThe adios group where the variables belongs
group_size_incThe inout adios group size to increment with the size of the variable
pathThe logical path structuring the data and containing the variable
nameThe variable name in the ADIOS file.
varThe variable to be defined. Used for type inference. Can be ignored.
Note
'name' and 'var' are written as successive arguments on purpose. One should be able to define a macro such as: #define STRINGIFY_VAR(x) #x, x Calling define_adios_double_scalar with such a macro will be done as: call define_adios_double_scalar(group, size, path, STRINGIFY_VAR(x)) as STRINGIFY_VAR(x) expand as: "x", x x being the variable name inside the code.
